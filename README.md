# Flower shop

Бот телеграм для заказа букетов из магазина цветов.

## Запуск

Для запуска блога у вас уже должен быть установлен Python 3.

- Скачайте код
- Установите зависимости командой `pip install -r requirements.txt`
- Запустите бота командой `python main.py`

## Переменные окружения

Часть настроек проекта берётся из переменных окружения. Чтобы их определить, создайте файл `.env` рядом с `main.py` и запишите туда данные в таком формате: `ПЕРЕМЕННАЯ=значение`.

Доступны следущие переменные:
- `POSTING_TELEGRAM_BOT_API_KEY` — Токен телеграм бота.
- `TOKEN_PAYMENT` — Токен платежной системы.
- `MANAGER_ID` — ID флориста в телеграм.
- `COURIER_ID` — ID курьера в телеграм..
